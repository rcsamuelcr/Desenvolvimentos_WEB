Funções Built-in do JavaScript

Um guia didático com explicações detalhadas e exemplos práticos.

Funções Globais

O que são:

Funções globais são utilitários já disponíveis em qualquer programa JavaScript — você não precisa importar nada para usá-las. No navegador, elas fazem parte do objeto global (window ou globalThis). São ferramentas “de uso geral” que facilitam tarefas comuns como:

  • Converter textos em números;
  • Testar valores;
  • Mostrar mensagens no console;
  • Codificar URLs.

As funções globais são:

console.log()

Exibe mensagens no console, ideal para testes e depuração de código.

console.log("Olá, mundo!"); // Saída: Olá, mundo!

parseInt()

Converte uma string em número inteiro. Caso encontre decimais, ignora a parte fracionária.

console.log(parseInt("10"));   // 10
      console.log(parseInt("10.5")); // 10

parseFloat()

Converte uma string em número decimal (mantendo a parte fracionária).

console.log(parseFloat("10.5")); // 10.5

isNaN()

Verifica se o valor informado não é um número.

console.log(isNaN("abc")); // true
      console.log(isNaN(123));   // false

isFinite()

Retorna true se o valor for um número finito.

console.log(isFinite(100));     // true
      console.log(isFinite(Infinity)); // false

encodeURI() e decodeURI()

Servem para codificar e decodificar URLs, garantindo que caracteres especiais sejam tratados corretamente.

let url = "https://site.com/página teste";
      let encoded = encodeURI(url);
      console.log(encoded); // https://site.com/p%C3%A1gina%20teste
      
      let decoded = decodeURI(encoded);
      console.log(decoded); // https://site.com/página teste

Funções de Manipulação de Arrays

O que é um arry:

Um array é uma coleção ordenada de valores (lista). Cada elemento tem uma posição (índice) que começa em 0: o primeiro item está em array[0], o segundo em array[1], etc. Pense num armário com gavetas numeradas — cada gaveta guarda um item.

Para que servem os métodos:

Os métodos dos arrays são funções prontas que permitem adicionar, remover, procurar e transformar os itens de forma prática, sem ter que escrever tudo do zero.

As funções de manipulação de arrays são:

push()

Adiciona elementos ao final do array.

let frutas = ["maçã", "banana"];
      frutas.push("laranja");
      console.log(frutas); // ["maçã", "banana", "laranja"]

pop()

Remove o último elemento do array.

frutas.pop();
      console.log(frutas); // ["maçã", "banana"]

shift()

Remove o primeiro elemento do array.

frutas.shift();
      console.log(frutas); // ["banana"]

unshift()

Adiciona elementos no início do array.

frutas.unshift("uva");
      console.log(frutas); // ["uva", "banana"]

map()

Cria um novo array aplicando uma função a cada elemento.

let numeros = [1, 2, 3];
      let dobro = numeros.map(n => n * 2);
      console.log(dobro); // [2, 4, 6]

filter()

Filtra os elementos que satisfazem uma condição.

let pares = numeros.filter(n => n % 2 === 0);
      console.log(pares); // [2]

reduce()

Reduz o array a um único valor, acumulando os resultados.

let soma = numeros.reduce((acc, n) => acc + n, 0);
      console.log(soma); // 6

Funções de Manipulação de Datas

O que são:

Date representa um ponto no tempo: data e hora. Internamente o JavaScript trabalha com milissegundos a partir de 1º de janeiro de 1970 UTC (o “epoch”), mas a API permite que você veja ano, mês, dia, hora local, etc.

Servem para:

  • Registrar quando algo aconteceu (ex.: hora de envio de um formulário).
  • Calcular diferenças entre datas (ex.: idade, tempo decorrido);
  • Exibir data/hora formatada para o usuário.

As funções globais são:

Date.now()

Retorna os milissegundos desde 1º de janeiro de 1970 (timestamp).

console.log(Date.now());

getFullYear()

Retorna o ano da data.

let hoje = new Date();
      console.log(hoje.getFullYear()); // 2025

getMonth()

Retorna o mês (0 = Janeiro, 11 = Dezembro).

console.log(hoje.getMonth());

getDay()

Retorna o dia da semana (0 = Domingo, 6 = Sábado).

console.log(hoje.getDay());

getHours()

Retorna a hora atual.

console.log(hoje.getHours());

Funções Matemáticas (Math)

O que são:

Math é uma coleção (namespace) de funções e constantes matemáticas. Não se instancia: você chama Math.sqrt(), Math.random() etc. Ele traz operações prontas para cálculos que seriam chatos de implementar.

Servem para:

  • Cálculos matemáticos em jogos, física, estatística, processamento (raiz, potência);
  • Arredondamentos (para mostrar valores com casas decimais);
  • Geração de números aleatórios (ex.: simuladores, jogos).

As funções matemáticas são:

Math.sqrt()

Calcula a raiz quadrada.

console.log(Math.sqrt(16)); // 4

Math.pow()

Calcula a potência (base, expoente).

console.log(Math.pow(2, 3)); // 8

Math.random()

Gera um número aleatório entre 0 e 1.

console.log(Math.random());

Math.floor()

Arredonda para baixo.

console.log(Math.floor(4.9)); // 4

Math.ceil()

Arredonda para cima.

console.log(Math.ceil(4.1)); // 5

Funções de Manipulação de Strings

O que é uma String:

Uma string é uma sequência de caracteres — texto. Em JavaScript, strings são imutáveis: qualquer método que “modifique” uma string na verdade cria uma nova string.

Para que servem os métodos de string:

  • Transformar (ex.: toLowerCase(), toUpperCase()), útil para comparar textos sem diferenciar maiúsculas/minúsculas;
  • Pegar pedaços (substring()), útil para extrair partes de uma entrada;
  • Substituir texto (replace()), validar formatos e limpar entradas do usuário;
  • Medir (length), importante para ver se a senha tem o tamanho mínimo etc.

toLowerCase()

Transforma todas as letras em minúsculas.

console.log("JavaScript".toLowerCase()); // javascript

toUpperCase()

Transforma todas as letras em maiúsculas.

console.log("JavaScript".toUpperCase()); // JAVASCRIPT

length

Retorna o número de caracteres da string.

console.log("Hello".length); // 5

substring()

Extrai uma parte da string.

let texto = "Programar é divertido";
      console.log(texto.substring(0, 9)); // Programar

replace()

Substitui parte de uma string por outra.

console.log("Olá, João".replace("João", "Maria")); 
      // "Olá, Maria"
